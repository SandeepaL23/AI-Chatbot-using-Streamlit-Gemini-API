# AI-Chatbot-using-Streamlit-Gemini-API
An interactive AI chatbot web application built with Streamlit and Google Gemini 1.5 Flash model. It provides real-time conversational responses, maintains chat history, and demonstrates integration of Generative AI into a simple and user-friendly web interface using Python.

## Project Overview
This is an interactive AI chatbot web application built using Streamlit and Google Gemini 1.5 Flash model.
It allows users to chat in real-time and get intelligent responses powered by Generative AI.

The project demonstrates how to integrate LLM APIs into a simple web interface using Python.

## Features
1. Real-time chat interface
2. Powered by Google Gemini 1.5 Flash
3. Maintains chat history during session
4. Fast and lightweight Streamlit UI
5. Simple and beginner-friendly implementation

## Tech Stack
1. Python
2. Streamlit - Web UI framework
3. Google Gemini API's
4. Gemini 1.5 Flash Model

## How it Works
1. User enters a message in the chat input
2. Message is sent to Gemini API
3. Model generates a response
4. Streamlit displays both user and AI messages in chat format
